openEO¶

openEO is an open API to express and execute (big) Earth observation algorithms and queries. As a common API it allows connecting different kind of client technologies (Python, R, JavaScript, …) with different computation back-ends in a simple and unified way.

  • see openeo.org for general information on openEO as an API

  • see openEO Platform (openeo.cloud) for a user-oriented and federated openEO platform
